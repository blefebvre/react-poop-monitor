# Poop Monitor

## About

baby scheduler

- poo
- pee
- eating
- waking up

track baby milestones
- smile 
- walk 
- talk

track gifts


## Run


$ babel --watch app-js-src/ --out-dir www/js/build/

in another term
$ cd www/
$ python -m SimpleHTTPServer
